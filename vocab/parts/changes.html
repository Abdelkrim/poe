Significant changes in this specification from the ODRL Community Group's draft include:

  • Clarified Permission cardinality in XML Encoding (Issue#17)
  • Added missing inheritRelation term (Issue#51)
  • Added compensatingParty term (Issue#53)
  • Added Assertion Policy Type (Issue#54)
  • Designated some terms as non-normative (Issue#64)
  • Added Right Operand Reference (Issue#56)
  • Added explict Left Operand (Issue#84)
  • Added inverse Target link (Issue#61)
  • Added UnitOfCount constraint (Issue#58)
  • Updated XML Encoding to support multiple and policy-level Assets, Parties, and Actions. (Issue#82)
  • Added Scopes to Assets and updated Scopes for Parties. (Issue#59)
